# Sistema de Turnos

## Descripción
Este proyecto es un sistema de gestión de turnos desarrollado en vanilla JavaScript como trabajo práctico para el curso de JavaScript de CoderHouse.  
Permite a los usuarios registrarse, iniciar sesión y administrar turnos de manera simple desde el navegador.

El sistema funciona completamente del lado del cliente utilizando LocalStorage para el almacenamiento de datos.

---

## Funcionalidades
- Registro de usuarios con validaciones
- Inicio y cierre de sesión
- Creación, edición y eliminación de turnos
- Búsqueda de turnos por servicio
- Estadísticas de turnos del usuario
- Visualización del turno más solicitado
- Carga dinámica de servicios desde un archivo JSON

---

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- Fetch API
- SweetAlert2

---

## Estructura del proyecto

- SISTEMATURNOS/
  - data/
    - servicios.json
  - index.html
  - main.js
  - styles.css
 

---

## Aclaraciones
- No utiliza base de datos ni backend.
- Los datos se almacenan en el navegador mediante LocalStorage.
- El inicio de sesión no es case sensitive para nombre y email.
- El proyecto está pensado con fines educativos.
